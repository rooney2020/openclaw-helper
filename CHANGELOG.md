# Changelog

## [0.1.0] - 2026-06-25

### Added
- 初始项目结构，Electron + Vue 3 + TypeScript
- 仪表板：Gateway 状态总览、渠道/Agent 概况
- 会话管理：会话列表浏览、搜索、删除
- 渠道管理：内置/插件渠道展示、配置编辑、启用/禁用、删除
- 配置管理：可视化编辑 Gateway 配置文件，自动备份
- 终端（TUI）：内嵌 xterm 终端
- 技能管理：查看已安装技能
- 用量统计：会话与 Agent 调用数据
- 笔记管理：工作区笔记 + 飞书知识库
- 日志查看

### Fixed
- 渠道列表仅显示已配置渠道 → 改为展示全部内置渠道，已配置/未配置分类
- 会话列表未显示文件名和文件大小 → 补充文件元数据
- 会话过滤"仅显示有对话"无效 → 读取文件头检测实际消息内容
- 仪表板 133 字节空会话展示 → 过滤掉无消息的会话文件
- `.trajectory.jsonl` 文件被误识别为会话 → 过滤轨迹文件
- 会话标题只显示 UUID → 改为 UUID + 首条用户消息双行展示

### Changed
- 渠道已配置检测逻辑：由特定字段检查改为配置对象非空判断
- 仪表板渠道卡片文字居中显示
- 已归档（compacted）会话合并到会话主列表，标记"归档"角标
- 会话列表后端加入 `hasMessages` 和 `title` 字段
